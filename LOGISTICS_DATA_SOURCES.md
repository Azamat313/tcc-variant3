## PART 2: OPEN APIs AND DATA SOURCES

### A. Vessel/Ship Tracking APIs

#### 1. MarineTraffic API
- **URL:** https://www.marinetraffic.com/en/ais-api-services
- **Docs:** https://servicedocs.marinetraffic.com/
- **Data:** Real-time AIS positions, vessel details, port calls, port congestion, voyage data
- **Pricing:** Paid (credit-based system, credits expire after 12 months)
- **REST API:** Yes (JSON/XML)
- **Embeddable:** Yes (widget available)
- **Notes:** Industry standard. Also offers Container Tracking API at container-tracking.marinetraffic.com

#### 2. VesselFinder API
- **URL:** https://api.vesselfinder.com/docs/
- **Data:** Real-time AIS positions, vessel particulars, voyage info
- **Pricing:** Paid (credit-based, from ~$50/month)
- **REST API:** Yes (JSON/XML)
- **Embeddable:** Yes (web widget)

#### 3. AISHub (FREE)
- **URL:** https://www.aishub.net
- **Data:** Real-time ship positions via AIS data exchange network
- **Pricing:** FREE (community data sharing model - you contribute AIS data, you get access)
- **REST API:** Yes (JSON/XML)
- **Embeddable:** No (data feed only)
- **Notes:** Best free option. Requires AIS receiver contribution or data share agreement

#### 4. Datalastic
- **URL:** https://datalastic.com
- **Docs:** https://datalastic.com/api-reference/
- **Data:** Real-time ship tracking, 750k+ vessels, AIS database
- **Pricing:** From EUR 199/month (credit-based)
- **REST API:** Yes (JSON)
- **Embeddable:** No (API only)

#### 5. Searoutes Vessel API
- **URL:** https://searoutes.com/vessel-api/
- **Data:** AIS data (terrestrial + satellite), updated every 5 min, route optimization
- **Pricing:** Freemium (limited free tier, paid for production)
- **REST API:** Yes
- **Embeddable:** No

---

### B. Flight / Air Cargo Tracking APIs

#### 1. OpenSky Network (FREE)
- **URL:** https://opensky-network.org
- **Docs:** https://openskynetwork.github.io/opensky-api/
- **Data:** Real-time ADS-B flight positions, state vectors, tracks, flight history
- **Pricing:** FREE for research/non-commercial. 4,000 API credits/day (8,000 for contributors)
- **REST API:** Yes (JSON) + Python/Java bindings
- **Embeddable:** Data can be used in custom visualizations
- **Notes:** Best free flight tracking API. No commercial flight schedules, only ADS-B derived data

#### 2. Flightradar24 API
- **URL:** https://fr24api.flightradar24.com
- **Data:** Real-time aircraft positions, historical tracks, airline/airport info
- **Pricing:** From $9/month (Explorer: 30k calls), $99/month (Essential: 450k calls), $900/month (Advanced: 4M calls)
- **REST API:** Yes (JSON)
- **Embeddable:** Yes (with licensing agreement)

#### 3. FlightAware AeroAPI
- **URL:** https://www.flightaware.com/commercial/aeroapi
- **Docs:** https://www.flightaware.com/aeroapi/portal
- **Data:** Flight status, tracking, airline/airport data, 60+ endpoints
- **Pricing:** Free trial tier available. Paid from $0.002/query
- **REST API:** Yes (JSON)
- **Embeddable:** Via data integration
- **Notes:** Good for air cargo tracking - supports logistics/cargo use cases

#### 4. AviationStack
- **URL:** https://aviationstack.com
- **Data:** Real-time flight data, airline routes, airport info
- **Pricing:** Free tier (100 requests/month), paid from $49.99/month
- **REST API:** Yes (JSON)
- **Embeddable:** Via API integration

#### 5. IATA Cargo Data (Reference)
- **URL:** https://www.iata.org/en/services/data/cargo/
- **Data:** Monthly air cargo traffic/capacity statistics, market intelligence
- **Pricing:** Paid subscription
- **REST API:** No (reports and datasets)
- **Embeddable:** No

### C. Freight Rates APIs

#### 1. Freightos Baltic Index (FBX) - FREE data
- **URL:** https://terminal.freightos.com
- **Developer Portal:** https://developers.freightos.com/freight-tools
- **Data:** Container freight rates on 12 major tradelanes, air freight indexes
- **Pricing:** Free account for viewing/downloading data. API access requires commercial agreement
- **REST API:** Developer portal available (requires sign-up)
- **Embeddable:** Charts downloadable as PNG. CSV export available
- **Notes:** IOSCO-compliant index. Updated weekly. The benchmark for container freight rates

#### 2. Drewry World Container Index (WCI)
- **URL:** https://www.drewry.co.uk/trackers-and-indices
- **Data:** Weekly spot container rates on 8 routes + composite index (USD/40ft)
- **Pricing:** Free weekly tracker on website. CFRI subscription for API access
- **REST API:** Available on request (paid)
- **Embeddable:** Free weekly chart available on website

#### 3. Baltic Exchange
- **URL:** https://www.balticexchange.com
- **Data:** Dry bulk (BDI), tanker, container freight indices
- **Pricing:** Paid subscription
- **REST API:** Via data partners (Thomson Reuters, Bloomberg)
- **Embeddable:** No

#### 4. ERAI Index 1520 (Eurasian Rail Rates)
- **URL:** https://index1520.com/en/index/
- **Data:** Rail container transit rates Russia-Belarus-Kazakhstan, forwarding/terminal costs
- **Pricing:** Free access to index data and statistics on website
- **REST API:** No documented public API
- **Embeddable:** No (but data freely viewable/downloadable from site)
- **Notes:** Critical for Middle Corridor rail rate analysis

#### 4. MarineTraffic Port Congestion

- **REST API:** Yes
- **Embeddable:** Via API

### E. Trade & Customs Data APIs

#### 1. UN Comtrade API (FREE)
- **URL:** https://comtradeplus.un.org
- **Developer Portal:** https://comtradedeveloper.un.org
- **Data:** International trade data, 200+ countries, annual (from 1988) + monthly (from 2000), goods + services
- **Pricing:** FREE (100k records/call, 500 API calls/day with free key)
- **REST API:** Yes (JSON) + Python library (comtradeapicall) + R package (comtradr)
- **Embeddable:** Data can be used in custom visualizations
- **Notes:** The gold standard for international trade data. Essential for trade flow analysis

#### 2. OEC API (Observatory of Economic Complexity)
- **URL:** https://oec.world/en/resources/api
- **Docs:** https://oec.world/en/resources/documentation
- **Data:** Trade data, economic complexity, 5000+ subnational regions, 5000+ products
- **Pricing:** Free basic access. Pro/Premium for full API and subnational data
- **REST API:** Yes (JSON)
- **Embeddable:** Yes (iframe embed codes for visualizations like treemaps)
- **Notes:** Excellent for visual trade data. Built by Datawheel/MIT

#### 3. World Bank WITS
- **URL:** https://wits.worldbank.org
- **Data:** Trade statistics, tariff data, non-tariff measures for all countries
- **Pricing:** FREE
- **REST API:** Limited (bulk download + some API endpoints)
- **Embeddable:** No

#### 4. WTO Global Trade Data Portal
- **URL:** https://globaltradedata.wto.org
- **Data:** Regional and global trade statistics, trade policy data
- **Pricing:** FREE
- **REST API:** Limited
- **Embeddable:** No

### G. Economic / Market Data APIs

#### 1. World Bank Indicators API (FREE)
- **URL:** https://api.worldbank.org/v2/
- **Docs:** https://datahelpdesk.worldbank.org/knowledgebase/articles/889392
- **Data:** 16,000+ indicators including GDP, trade, LPI (Logistics Performance Index), infrastructure
- **Pricing:** FREE (no API key needed)
- **REST API:** Yes (JSON, XML, CSV, Excel)
- **Embeddable:** Data freely usable
- **Example:** `https://api.worldbank.org/v2/country/KAZ/indicator/LP.LPI.OVRL.XQ?format=json`
- **Notes:** Essential. Includes LPI for Kazakhstan, Azerbaijan, Georgia, Turkey, China

#### 2. World Bank Logistics Performance Index (LPI)
- **URL:** https://lpi.worldbank.org
- **DataBank:** https://databank.worldbank.org/source/logistics-performance-index-(lpi)
- **Data:** LPI scores for 139 countries, customs efficiency, infrastructure quality, tracking capability
- **Pricing:** FREE
- **REST API:** Via World Bank API (indicator code: LP.LPI.OVRL.XQ)
- **Embeddable:** Data freely usable
- **Notes:** Benchmarks logistics performance. Critical for Middle Corridor country comparisons

#### 3. IMF Data API (FREE)
- **URL:** https://data.imf.org/en
- **API Page:** https://data.imf.org/en/Resource-Pages/IMF-API
- **Data:** World Economic Outlook, commodity prices, trade, inflation, fiscal indicators
- **Pricing:** FREE
- **REST API:** Yes (SDMX 2.1 and 3.0 format)
- **Embeddable:** Data freely usable

#### 4. Alpha Vantage (FREE tier)
- **URL:** https://www.alphavantage.co
- **Data:** Commodities (oil, gas, metals), forex, stock data
- **Pricing:** Free (25 API calls/day), paid for higher volume
- **REST API:** Yes (JSON, CSV)
- **Embeddable:** Via API integration

#### 5. Commodities-API
- **URL:** https://commodities-api.com
- **Data:** Real-time and historical commodity prices (oil, metals, agricultural)
- **Pricing:** Freemium (limited free tier)
- **REST API:** Yes (JSON)
- **Embeddable:** Via API

#### 6. Trading Economics API
- **URL:** https://tradingeconomics.com/api/
- **Data:** Exchange rates, stock indexes, commodity prices, government bonds, economic indicators
- **Pricing:** Paid (from $49/month)
- **REST API:** Yes (JSON)
- **Embeddable:** Via API

## PART 3: CENTRAL ASIA / MIDDLE CORRIDOR SPECIFIC DATA

### TITR Association
- **Official Site:** https://middlecorridor.com/en/
- **Data Available:** Freight volumes (4.5M tons/year via KZ, 70% growth in 2024), transit times (18-23 days, target 14-18), container train counts
- **API:** None. Data via reports and press releases
- **Key Stats:** Target 600 container trains/year in 2025-26, growing to 2,000 by 2029

### KTZ (Kazakhstan Railways)
- **Data via:** https://index1520.com/en/ (ERAI platform)
- **Khorgos-Altynkol:** ~14M tons handled in 2024. Transshipment 24-36 hours (1435mm to 1520mm gauge change)
- **API:** No public API. Statistics available via Index 1520 and government reports
- **Key Info:** 2,000+ km rail infrastructure upgrades connected to Middle Corridor

### Aktau Port
- **Operator Data:** Via KZ government and TITR reports
- **Throughput:** ~12M tons capacity. 2M tons in H1 2023 (+23%)
- **API:** None
- **Key Info:** Wind-resistant ship-to-shore cranes being installed

### Kuryk Port
- **Official Site:** https://kuryk.kz/en/
- **Data:** 1.96M tons in 2024. Target 2.45M tons in 2025 (+25%)
- **Breakdown (2024):** Rail: 1.15M tons, Road: 781k tons, Grain terminal: 25.9k tons
- **API:** None
- **Key Info:** Depth increased from 5.8m to 7.5m in 2024. New grain terminal operational since Nov 2024

### Azerbaijan Railways (ADY)
- **Official Site:** https://corp.ady.az/en
- **Data:** 18M+ tons general cargo in 2023, ~7M tons transit (+80% growth)
- **BTK Railway:** Baku-Tbilisi-Kars expansion from 1M to 5M tons capacity
- **API:** None
- **Transit Time:** China to Azerbaijan via Middle Corridor: 8-10 days

### Georgian Railway
- **Official Site:** https://www.railway.ge/en/
- **Data:** Part of KTZ-ADY-GR joint venture for Middle Corridor development
- **API:** None
- **Key Info:** BTK railway Georgian segment expansion underway

### TRACECA
- **Official Site:** https://traceca-org.org/en/
- **Data:** 14 member countries, corridor freight volumes (growing 4-5%/year on average)
- **API:** None. Reports and data via website
- **Secretariat:** Based in Baku, Azerbaijan
